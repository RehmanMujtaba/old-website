Autonomous Drone Flightmore_vert

Python, Sockets, Electronics

Autonomous Drone Flightclose Python

I soldered together a quadcopter using parts I ordered online. Once I was satisfied with a simple remote controlled drone, I got hold of a Raspberry Pi which was able to connect to my flight controller chip. From there, I was able to establish a Wifi connection between my laptop and the drone (so much for all the money spent on a dedicated transmitter/receiver). After that, using the DroneKit Python API, I was able to code the drone for semi-autonomous flight and ariel tricks! Of course, first it was better to keep the drone on a leash (yes, a physical leash), because a small mistake could always lead to the drone flying away into the sky!

Command Line Text Editormore_vert

C, Bash, Linux

Command Line Text Editorclose C

I am currently working on a command line text editor for Linux that I am making in C. Before this project, the terminal was like a black-box to me. You type in some text and it does what you ask it to (more often dose not since you spelt something wrong). But now, I feel like I really appreciate whats going on under the hood. The terminal listens to each and every key press carefully, especially watching out for ctrl+anything and complicated escape sequences (more of a C thing, but let me have this). Another thing I enjoy about this project is how extendable it is. I can add as many features as I want: syntax highlighting, searching and fonts to name a few.
